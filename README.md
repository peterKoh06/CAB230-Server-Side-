After downloading files, You need to install modules to run it and test it. 

1) Open the terminal and command "NPM INSTALL"
2) Once you have downloaded modules, you need directory toward rentalapi-tests file to test it.
3) Open the terminal separately, either left or right command "cd rentalapi-tests"
4) You need to command two different commands at the same time. For example, left side "node app.js" for running the server and on right right, you want to type "npm test" to test your project how many tests you could pass.
